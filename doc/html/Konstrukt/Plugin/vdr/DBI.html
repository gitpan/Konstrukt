NAME

Konstrukt::Plugin::vdr::DBI - VDR Backend Driver

SYNOPSIS

        #TODO

DESCRIPTION

The DBI backend for the personal userdata.

METHODS

init

Initialization of this class

check_collision

Returns the id of a colliding show if there is a collision with schedules shows.

Parameters:

  • $start_year, $start_month, $start_day, $start_hour, $start_minute - Start date of the new entry
  • $end_year, $end_month, $end_day, $end_hour, $end_minute - End date of the new entry

add_entry

Adds a new tv show.

Parameters:

  • $channel - The name of the TV channel
  • $title - The title of the show
  • $year, $month, $day - The start date of the show
  • $start_hour, $start_minute - The start time
  • $end_hour, $end_minute - The ending time
  • $jobs - The atd jobs that will start the recording progress

get_entries

Returns an array of hash references with the keys id, channel, title, year, month, day, start_hour, start_minute, end_hour, end_minute, jobs.

get_entry

Returns one specified saved tv show as an hash references with the keys id, channel, title, year, month, day, start_hour, start_minute, end_hour, end_minute, jobs

Parameters:

  • $id - The id of the show

delete_entry

Removes an existing entry.

Parameters:

  • $id - The id of the entry, which should be removed
